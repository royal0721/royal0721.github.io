閉包

  • 函式能夠存取的範疇，即是其被內嵌後往外推的範圍。
  • Javascript 垃圾回收機制會釋放不再用的記憶體，但是在閉包因為目的在於能存取語言範疇，因此在是不再釋放不用的記憶體，會予以保留，不做記憶體回收。

內層函式由於具有閉包的特性，因此可存取外層的變數和函式（something 與 another）。透過模組模式，可隱藏私密資訊，並選擇對外公開的API。

管理多個模組-模組依存性載入器（module dependency loader）

  • define module

    bind vs. call vs. apply

  • bind: 將函數綁定到一個對象上，返回一個函數。調用返回的函數，會將原函數当做绑定的對象的方法來使用。
    Binding something in JavaScript means recording that identifier(function,param) in a specific Environment Record(可以理解為相應代碼塊內的所有變量聲明、函數聲明（代碼塊若為函數還包括其形參）都儲存於此對應ES6之前的變量對象or活動對象). Each Environment Record is related to a specific Execution Context - and that binds the identifier (variable or function name) to the this keyword for that execution context.
